{% extends "base.html" %} {% block title %}About the footbag database{% endblock %} {% block header %}About this site{% endblock %} {% block content %} Footbag freestyle is a technically challenging sport. Being able to find quality information when you need it is therefore very important.

Until now if you didn't know the names of various moves it was hard to find any information. With the development of the footbag DB you have better ways to search for footbag related information.

By creating a database of footbag moves we are able to organize the information relating to those moves a lot better.

The code powering this site can be found over on the github project page. This is open source code so do check it out if you are technially inclined, bug reports and patches are welcome!
